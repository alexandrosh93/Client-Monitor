Private GitHub Gist — no app registration needed

  1. Go to github.com → sign in (or create free account)
  2. Click your profile picture → Settings → scroll to Developer settings
  3. Personal access tokens → Tokens (classic) → Generate new token
  4. Give it any name, tick only gist, click Generate token
  5. Copy the token and paste it below — the app creates a private gist automatically
